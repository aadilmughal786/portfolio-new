# Styling with Tailwind CSS

Tailwind CSS is a utility-first CSS framework that speeds up styling.

## Why Tailwind?

- **Utility Classes**: Apply styles directly in HTML/JSX.
- **Responsive Design**: Built-in support for mobile-first layouts.
- **Customizable**: Extend with custom themes.

## Example: Card Component

```tsx
<div className="p-6 bg-white rounded-lg shadow-md">
  <h2 className="text-xl font-bold">Card Title</h2>
  <p className="mt-2 text-gray-600">This is a card description.</p>
</div>
```

## Tips

Purge unused styles in `tailwind.config.js` to reduce bundle size.
