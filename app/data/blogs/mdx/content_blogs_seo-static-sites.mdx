# SEO for Static Sites

Search engine optimization (SEO) ensures your site ranks well on Google.

## Techniques

- **Metadata**: Use `generateMetadata` for dynamic titles and descriptions.
- **Sitemaps**: Generate a `sitemap.xml` for crawlers.
- **Fast Loading**: Static sites load quickly, boosting SEO.

## Example: Metadata

```tsx
export async function generateMetadata() {
  return {
    title: 'My Blog Post',
    description: 'A blog post about SEO.',
  };
}
```

## Tools

Use [Google Search Console](https://search.google.com/search-console) to monitor performance.
