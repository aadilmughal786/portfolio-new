# Deploying Static Sites to GitHub Pages

GitHub Pages is a free hosting solution for static sites. Here's how to deploy a Next.js site.

## Steps

1. Configure `next.config.js` with `output: 'export'`.
2. Set `basePath` and `assetPrefix` for subdirectories.
3. Run `npm run build` to generate the `out/` directory.
4. Deploy `out/` to the `gh-pages` branch using `gh-pages`.

## Example Config

```js
const nextConfig = {
  output: 'export',
  basePath: '/portfolio-new',
  assetPrefix: '/portfolio-new',
};
```

## Troubleshooting

Ensure all asset paths include the `basePath`.
