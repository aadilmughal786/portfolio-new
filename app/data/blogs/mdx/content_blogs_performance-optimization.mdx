# Optimizing Performance in Next.js

Performance is critical for user experience and SEO.

## Strategies

- **Static Generation**: Use `generateStaticParams` for pre-rendering.
- **Image Optimization**: Set `images.unoptimized: true` for static exports.
- **Minification**: Enable `swcMinify` in `next.config.js`.

## Example: Static Params

```tsx
export async function generateStaticParams() {
  return [{ slug: 'post1' }, { slug: 'post2' }];
}
```

## Tools

Use [Lighthouse](https://developers.google.com/web/tools/lighthouse) to audit performance.
